_What's That Chord Doing_? is a single-page app made with React and Redux.

Dependencies:

* React

* React DOM

* React Router 3

* React Redux 

* Redux Thunk

* [Vexflow](https://github.com/0xfe/vexflow/), for notation

* [Soundfont-player](https://github.com/danigb/soundfont-player), for our audio

* Babel
